# Halloween PropGen

A prop idea generator for Halloween haunts. Inspire your spooky heart.

## Requirements

* JDK 17
* Maven

## Set up

Clone the git repo:

`git clone git@github.com:ruscoe/halloween-propgen.git`

Run install:

`mvn clean install`

## Running

Start the server:

`mvn clean spring-boot:run`

Open: `http://localhost:8080/api/prop`
